# Profile Card

A premium, responsive profile card built with HTML, CSS, and Font Awesome. This project features a clean glassmorphism design, mobile compatibility, and social media integration.

## Features

- **Glassmorphism Design**: Modern UI with backdrop blur and semi-transparent backgrounds.
- **Responsive Layout**: Optimized for both desktop and mobile devices.
- **Font Awesome Icons**: High-quality, scalable social media icons.
- **Interactive Elements**: Smooth hover effects on the card, social links, and skill tags.
- **Portfolio Link**: A direct call-to-action button linking to the developer's full portfolio.

## Tech Stack

- **HTML5**: For structured semantic content.
- **CSS3**: For advanced styling, positioning, and animations.
- **Font Awesome**: For vector social icons.
- **Google Fonts**: 'Poppins' for clean typography.
